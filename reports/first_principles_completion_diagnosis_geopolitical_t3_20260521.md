## 1. Context（背景與提案動機）

### 1.1 對話脈絡

2026-05-21 與使用者對話中，分析 §0.1 第一性原理之 F（資訊力）變數時，揭露 §0.1 原版定義之 F 涵蓋 **4 個內涵維度**：

1. 盈餘超預期（Earnings Surprise）
2. 宏觀數據衝擊（Macro Shock）
3. **地緣 / 政策意外**（Geopolitical / Policy Surprise）
4. 訂單流（Order Flow）

其中第 1, 2, 4 維皆有現行 13 表 DATASET_REGISTRY 對應之 proxy（雖完整性各異），但第 3 維 **「地緣 / 政策意外」在現行 13 表中完全沒有對應**，且：

- FinMind API 不提供結構化新聞 / 事件資料
- 需自建 NLP 新聞 pipeline 才能抽取
- TWSE / TPEx 亦無原生結構化政策事件 feed

### 1.2 使用者裁決

使用者問：**「地緣 / 政策意外 ❌ 完全沒有，這個可以排除嗎？」**

研究三種方案後（A. 完全移除概念 / B. 降階為 T3 永禁 / C. 保留為 T2 不實作），使用者選擇**方案 B**。

### 1.3 為何此提案重要

「地緣 / 政策意外」目前處於**治權真空**：
- §0.1 概念清單仍列為 F 之內涵 → 暗示有實作義務
- 但沒有可觀測 proxy → 永遠無法實作
- 也沒有明文 §0.1-A 禁令 → 後繼者可能誤建「新聞情緒因子」並硬塞進 prediction layer

**方案 B 的本質**：把現狀（永遠無法實作）**明文治權化**，類比 IFF Θ / SOC / 重力井邊緣之既有 T3 處理。

### 3.4 §0.1-E 證偽承諾——是否需修改？

**裁決**：**不需修改**。

理由：現行 §0.1-E 之 **T3.1 條款**為：

> T3.1 IFF Θ / SOC 之可實作性檢驗：任何提案實作 IFF Θ 或 SOC 須先通過 5 年回測證明 IC ≥ rank-IC baseline；若無實作通過此檢驗，T3 元素永久維持「隱喻層」不入 model

此條款的精神已涵蓋「**任何 T3 元素若有未來實作提案，須通過 5 年回測**」之普遍性原則。地緣 / 政策意外作為新增 T3 元素自動繼承此條款，**無需新增獨立證偽指標**。

若要更顯式，可選擇性把 T3.1 標題改為「IFF Θ / SOC / **地緣事件 NLP** 之可實作性檢驗」（**選用**，非必須）。
